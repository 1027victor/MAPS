# MAPS-Explore — User Guide

MAPS-Explore is a practical 3D visualization tool for spatial transcriptomics. It covers project management, data management, 2D visualization, slice alignment, 3D visualization, and more. Before using the tool, install and launch the application, then access the front-end through your browser.
